# Placetrix

> **The Gap Between You and Your Goal? Let's Close It.**

Placetrix is an educational assessment and placement platform designed to streamline aptitude preparation, programming assessments, job listings, and campus placement drives. By providing distinct, feature-rich dashboards for **Candidates**, **Institutions**, and **Recruiters**, Placetrix bridges the gap between students, educators, and employers.
